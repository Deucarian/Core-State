# Support for Deucarian Core State

Use [GitHub Issues](https://github.com/Deucarian/Core-State/issues) for reproducible repository or selection-service defects, documentation gaps, and narrowly scoped feature requests. Search existing issues before opening a new one.

Include the package version, Unity editor version, the smallest state/repository example that reproduces the behavior, expected and actual results, and the full error or stack trace.

`main` is the stable package channel. `develop` is a preview channel and may change before promotion. Please reproduce against the applicable current channel before reporting an older snapshot.

Do not disclose vulnerabilities or private project data in an issue. Follow [SECURITY.md](SECURITY.md) for private security reports.

Support is provided on a best-effort basis. Domain-specific state systems and Unity editor support outside this package's repository/selection boundary are out of scope.
